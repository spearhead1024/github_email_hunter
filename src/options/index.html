GitHub Email Hunter

This extension finds personal email addresses already public in GitHub commit metadata. All data stays in your browser; nothing is sent anywhere except to api.github.com.

Personal Access Token (optional)

Without a token, GitHub allows only 60 API requests/hour. With a token (any classic or fine-grained PAT, no scopes required), the limit becomes 5,000/hour.

Create one at github.com/settings/tokens. No scopes are needed for public-data reads.

Cache

Scan results are cached locally for 24 hours.

Privacy & responsible use

Commit author emails are technically public — they live inside git log for every public repository on GitHub. This extension surfaces what's already there. Even so, please use it responsibly: do not bulk-scrape, do not spam, and if a user has set a privacy-proxy (users.noreply.github.com), respect it.

To hide your own email, enable "Keep my email addresses private" in your GitHub email settings, and configure your local git config user.email to match the proxy address GitHub provides.
