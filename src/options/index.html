GitHub Email Hunter

All data stays in your browser; nothing is sent anywhere except to api.github.com.

Personal Access Tokens

Without a token, GitHub allows only 60 API requests/hour. With a token (any classic or fine-grained PAT, no scopes required), 5,000/hour.

⚠ About multiple tokens: GitHub rate-limits per account, not per token. Two PATs from the same GitHub account share the same 5,000/hr bucket — they give you no extra throughput. To actually parallelize a crawl, you need PATs from different GitHub accounts (e.g. teammates contributing their own tokens). Creating additional accounts solely to bypass rate limits violates GitHub's Acceptable Use Policy.

Cache

Single-profile lookup results are cached locally for 24 hours.

Privacy & responsible use

Commit author emails are technically public. Even so, please use this tool responsibly: do not bulk-spam, do not redistribute the data, and respect users who set the GitHub privacy proxy (users.noreply.github.com). Bulk collection of EU-resident contact data is regulated by GDPR.
